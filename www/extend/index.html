Check out and build

  1. Log into the java.net CVS repository using the following command:

    cvs -d :pserver:@cvs.dev.java.net:/cvs co login
  2. Check out the sources using:

    cvs -d :pserver:@cvs.dev.java.net:/cvs co wsmonitor/wsmonitor
  3. Build the workspace using:

    cd wsmonitor/wsmonitor
    mvn assembly:assembly
